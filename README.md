## 프로젝트 명세서
### 1. 프로젝트 기능

- **사용자 -** 회원가입 및 로그인, 회원정보 수정, 회원가입시 유효성 검사 및 중복 검사
- **게시판 -** 게시판 CRUD 기능, 조회수, 댓글 및 답글 수, 페이징 및 검색 처리
- **댓글 및 답글 -** 댓글 및 답글(대댓글) CRUD 기능
